扩展风控示例 — 三种新规则类型
复杂交叉表 多维场景定价矩阵 (RC_MULTI_DIM_RATE)
复杂交叉表：支持多维行列交叉 + 区间匹配。本示例为 4 维交叉： 业务类型 × 结算方式 × 客户类型 × 纳税人资格， 在多维矩阵中查找适用的组合定价系数。
复杂评分卡 交易票据异常评分 (RC_INVOICE_FRAUD_SCORE)
复杂评分卡：支持分组维度 + 结构化条件。评分维度按组分类： 客户基础信息（客户等级 + 月消费金额）、 发票合规指标（开票偏差率 + 红冲比例）、 合规风险指标（零税率占比 + 跨地区开票比例）， 同维度内互斥取分，维度间累加，最后按阈值输出风险等级。
QL脚本 混业组合计费脚本 (RC_BLEND_CALC_SCRIPT)
QL脚本：技术人员直接编写 QLExpress 脚本。本示例演示低费率产品线(9%)与高费率产品线(6%)按比例分拆计费， 支持小规模纳税人简易计税(3%)。脚本直接存储并执行，无需可视化设计器。
对象传入 对象传入定价 (RC_PRICING_BY_OBJECT)
对象传入说明：将多个参数封装为 ctx 对象传入规则引擎，规则脚本通过 taxpayerType、goodsCategory 访问属性。 对应数据对象定义：TaxContext（rule_data_object 表）。
三种函数类型演示
SCRIPT roundTax（经 RC_EXPOSURE_FLOW 演示）
SCRIPT 函数：服务端发布规则时自动将 SCRIPT 函数定义拼进 compiledScript， 客户端同步后直接执行，无需额外注册。同时 SDK 启动时也会通过 addFunctionsDefinedInScript 注册。 RC_EXPOSURE_FLOW 决策流的「计算税额」节点调用了 roundTax(taxAmount)。
JAVA JAVA 函数试算 (RC_FLOW_JAVA_SAMPLE) — calculateVAT
JAVA 函数：SDK 启动时从服务端同步函数定义，根据 implClass 反射实例化 com.bjjw.rule.example.functions.TaxFunctions，自动注册 calculateVAT 到本地引擎。 规则脚本中写 taxAmount = calculateVAT(totalAmount, taxRate)，在客户端本地执行。
BEAN Spring Bean 试算 (RC_FLOW_BEAN_SAMPLE) — calcTaxByBean
BEAN 函数：SDK 启动时从服务端同步函数定义，根据 implBeanName 从 Spring 容器获取 taxFunctions Bean，自动注册 calcTaxByBean 到本地引擎。 规则脚本中写 taxAmount = calcTaxByBean(totalAmount, taxRate)，在客户端本地执行。
